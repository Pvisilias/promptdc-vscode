# Contributing

Thanks for your interest in PromptDC (VS Code extension).

## What We Accept
- Bug fixes
- Documentation improvements
- Small UX or wording improvements

## How to Contribute
1. Fork the repo
2. Create a branch for your change
3. Make your changes with clear commit messages
4. Open a pull request with:
   - What changed
   - Why it helps
   - Screenshots (if UI changes)

## Reporting Issues
If you find a bug, open an issue with steps to reproduce and environment details.

## Security
If you discover a security issue, email **spromptdc@gmail.com**.
